# ⚡ Instalação Rápida do APK

## Para Usuários (Sem Programação)

### 1️⃣ Receber o APK
- Baixe o arquivo `seringal-v1.0.0.apk` via:
  - Google Drive / Dropbox
  - Email
  - WhatsApp
  - Link direto

### 2️⃣ Preparar o Telefone
1. Abra **Configurações** 
2. Vá para **Segurança** ou **Privacidade**
3. Ative "Fontes Desconhecidas" ou "Instalar aplicativos de fontes desconhecidas"

### 3️⃣ Instalar o App
1. Abra o arquivo `.apk` (Gerenciador de Arquivos → Downloads)
2. Toque em **"Instalar"**
3. Aguarde a instalação (2-5 segundos)
4. Toque em **"Abrir"** quando concluído

### 4️⃣ Usar o App
1. Faça login com credenciais fornecidas
2. Navegue livremente pelos módulos
3. **Funciona 100% offline** - sem internet necessária!
